# Twinstor
Twinstor is a new Layer 1 blockchain that combines high-speed transactions with native AI inference. Unlike every other blockchain today, Twinstor lets smart contracts call a large language model directly on-chain — and the result is cryptographically verified, not just trusted.

It runs at 5,200+ transactions per second with sub-second finality, is fully compatible with Ethereum(EVM-compatible), and uses state-of-the-art open-source AI models including DeepSeek-V3.2 and GLM-5. Our local simulator has validated the core architecture.
The roadmap targets a public testnet in Q2 2026 and mainnet launch in Q4 2026.
